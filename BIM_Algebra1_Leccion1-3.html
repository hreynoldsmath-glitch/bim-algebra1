  • Identificar ecuaciones con sin solución (no solution).
  • Identificar ecuaciones con infinitas soluciones (infinitely many solutions).
  • Aplicar la estrategia de las 4 etapas a problemas de la vida real.
📚 Core Vocabulary
⭐ Core Concept — Variables en Ambos Lados

Estrategia: Mueve todos los términos con variable a un lado y todas las constantes al otro. Luego resuelve como siempre.

3x + 2 = x + 8 → 3x − x = 8 − 2 → 2x = 6 → x = 3
Sin Solución / No Solution
3 = 7 (falso)
Las variables se cancelan → constantes desiguales
Infinitas Soluciones
0 = 0 (siempre verdadero)
Las variables se cancelan → identidad verdadera
🧩 Estrategia de 4 Pasos — Variables en Ambos Lados
1.
Simplifica
Distribuye y combina términos semejantes.
2.
Mueve x
Agrega/resta términos con variable a un lado.
3.
Despeja
Suma/resta constantes, luego divide.
4.
Verifica
Sustituye en la original.
Parte 1 — Variables en Ambos Lados (básico)
1

Ejemplo 1 — Resolver con Variables en Ambos Lados

5x − 4 = 2x + 11
Resuelve: 5x − 4 = 2x + 11
5x − 4 = 2x + 11Ecuación original
5x − 2x − 4 = 11Resta 2x de ambos lados (mueve x al izquierdo)
3x − 4 = 11Simplifica: 5x − 2x = 3x
3x = 15Suma 4 a ambos lados
x = 5Divide entre 3
✓ Verifica: 5(5)−4 = 25−4 = 21  |  2(5)+11 = 10+11 = 21 ✓
2

Ejemplo 2 — Con Distributiva y Variables en Ambos Lados

3(1 + x) = −2(x − 4)
Resuelve: 3(1 + x) = −2(x − 4)
3(1 + x) = −2(x − 4)Ecuación original
3 + 3x = −2x + 8Aplica Propiedad Distributiva
3 + 3x + 2x = 8Suma 2x a ambos lados
3 + 5x = 8Combina: 3x + 2x = 5x
5x = 5Resta 3 de ambos lados
x = 1Divide entre 5
✓ Verifica: 3(1+1)=3(2)=6  |  −2(1−4)=−2(−3)=6 ✓

Parte 2 — Soluciones Especiales
Casos especiales: Cuando resuelves y las variables desaparecen, el resultado te dice:
• Identidad falsa (ej: 3 = 7) → Sin Solución (ecuación inconsistente)
• Identidad verdadera (ej: 0 = 0) → Infinitas soluciones (ecuación idéntica)
3

Ejemplo 3 — Sin Solución / No Solution

Resuelve: 3(5x − 4) = 15x + 2
3(5x − 4) = 15x + 2Ecuación original
15x − 12 = 15x + 2Distribuye: 3·5x=15x, 3·(−4)=−12
−12 = 2Resta 15x de ambos lados
❌ −12 = 2 es una declaración falsa. La ecuación no tiene solución (no solution). No existe ningún valor de x que haga la ecuación verdadera.
4

Ejemplo 4 — Infinitas Soluciones

Resuelve: −2(x − 3) = −2x + 6
−2(x − 3) = −2x + 6Ecuación original
−2x + 6 = −2x + 6Distribuye: −2·x=−2x, −2·(−3)=+6
0 = 0Suma 2x, resta 6 — todo se cancela
✅ 0 = 0 es siempre verdadero. La ecuación tiene infinitas soluciones. Cualquier valor de x satisface la ecuación, ya que ambos lados son idénticos.

Parte 3 — Aplicaciones de la Vida Real
5

Ejemplo 5 — Cinta de Medición / Yardstick Problem

Dos yardsticks: Uno de 4x − 3 pies y otro de x + 12 pies. ¿Para qué valor de x tienen la misma longitud?
4x − 3 = x + 12Establece las expresiones iguales
3x − 3 = 12Resta x de ambos lados
3x = 15Suma 3 a ambos lados
x = 5Divide entre 3
✓ Longitud: 4(5)−3 = 17 pies  |  5+12 = 17 pies ✓
🔍

Monitoring Progress 3 — Vida Real

⚠️ Error Analysis
✗ Encuentra el error
5x + 3 = 3x − 7 5x + 3x = −7 + 3 ← ¿Correcto? 8x = −4 x = −1/2 ✗
Error: Al pasar 3x al izquierdo debería restar (5x − 3x = 2x), no sumar. Y −7 permanece en el derecho.
Correcto: 5x − 3x = −7 − 3 → 2x = −10 → x = −5
✓ 5(−5)+3 = −22  |  3(−5)−7 = −22 ✓
