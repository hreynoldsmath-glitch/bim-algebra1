⭐ Core Concept — Reescribir Ecuaciones Literales

Una ecuación literal (literal equation) tiene más de una variable. Para despejar una variable, usa las mismas propiedades de igualdad (suma, resta, multiplicación, división) — trata las otras variables como constantes.

Ejemplo: A = lw → despeja l → l = A/w
Parte 1 — Despejar una Variable en una Ecuación Lineal
Recuerda: "Despejar" significa aislar una variable en un lado. Aplica operaciones inversas igual que en ecuaciones de una variable, pero trata las demás variables como si fueran números.
1

Ejemplo 1 — Reescribir ecuación lineal

−5x + y = 3 (despeja y)
Resuelve para y: −5x + y = 3
−5x + y = 3Ecuación original
−5x + y + 5x = 3 + 5xSuma 5x a ambos lados
y = 5x + 3Simplifica — y está despejada
✓ Esta forma es útil para graficar: pendiente m = 5, intersección b = 3.
2

Ejemplo 2 — Despejar x

3x − 2y = 12 (despeja x)
Resuelve para x: 3x − 2y = 12
3x − 2y = 12Ecuación original
3x = 2y + 12Suma 2y a ambos lados
x = (2y + 12) / 3Divide entre 3
x = (2y)/3 + 4Simplifica (opcional)

Parte 2 — Despejar Variables en Fórmulas
💡 Estrategia: Trata la variable a despejar como si fuera una "x" normal. Todas las demás letras se comportan como números. Aplica operaciones inversas en orden inverso PEMDAS.
3

Ejemplo 3 — Fórmula d = rt (Despejar r)

La fórmula de distancia es d = rt. Despeja r (rata / velocidad).
d = rtFórmula original
d/t = rt/tDivide entre t en ambos lados
r = d/tSimplifica — r está despejada
✓ Aplica: Si d = 150 millas y t = 3 horas, entonces r = 150/3 = 50 mph.
4

Ejemplo 4 — Temperatura F a C

La fórmula es F = (9/5)C + 32. Despeja C (temperatura Celsius).
F = (9/5)C + 32Fórmula original
F − 32 = (9/5)CResta 32 de ambos lados
(F − 32) × (5/9) = CMultiplica por 5/9 (inverso de 9/5)
C = (5/9)(F − 32)C está despejada
✓ Verifica: Si F = 212°F → C = (5/9)(212−32) = (5/9)(180) = 100°C ✓ (punto de ebullición del agua)
5

Ejemplo 5 — Área de Triángulo

La fórmula del área de un triángulo es A = (1/2)bh. Despeja h (altura).
A = (1/2)bhFórmula original
2A = bhMultiplica por 2 ambos lados
h = 2A/bDivide entre b
✓ Aplica: Si A = 24 cm² y b = 8 cm, entonces h = 2(24)/8 = 6 cm.

Ejemplo 6 — Interés Simple (Despejar P)

La fórmula I = Prt. Si el interés es $525, la tasa es 3.5% (0.035) por año, y el tiempo es 4 años, ¿cuánto es el principal P?
I = PrtFórmula original
P = I/(rt)Despeja P: divide entre rt
P = 525/(0.035 × 4)Sustituye valores
P = 525/0.14Simplifica denominador
P = $3,750Solución
✓ Verifica: I = 3750 × 0.035 × 4 = 3750 × 0.14 = $525 ✓
🔍

⚠️ Error Analysis
✗ Encuentra el error — Despejar h de V = lwh
V = lwh V − l − w = h ← Error h = V − l − w ✗
Error: l y w están multiplicando h (no sumando). Para despejar h, hay que dividir entre lw, no restar l y w.
Correcto: V = lwh → V/(lw) = h → h = V/(lw)
✓ Aplica: Si V=60, l=4, w=3: h = 60/(4·3) = 60/12 = 5 ✓
