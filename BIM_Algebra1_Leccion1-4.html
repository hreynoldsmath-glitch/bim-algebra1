🎯What You Will Learn / Lo que aprenderás
  • Entender el significado de valor absoluto como distancia desde cero en la recta numérica.
  • Resolver ecuaciones de valor absoluto usando dos casos (positivo y negativo).
  • Identificar ecuaciones con sin solución (cuando el valor absoluto = número negativo).
  • Aplicar el valor absoluto a situaciones de la vida real como tolerancias y errores.
📚 Core Vocabulary
⭐ Core Concept — Valor Absoluto / Absolute Value

|x| = la distancia de x al cero en la recta numérica. Siempre es ≥ 0.

|5| = 5  ·  |−5| = 5  ·  |0| = 0

|x| = a (a > 0)
Caso +: x = a
Caso −: x = −a
↔
|x| = 0
Solución única:
x = 0
↔
|x| = a (a < 0)
Sin solución
(no solution)
Parte 1 — Resolver Ecuaciones de Valor Absoluto
Regla de los Dos Casos: Si |expresión| = c y c ≥ 0, entonces:
→ Caso + : expresión = c    O    Caso − : expresión = −c
Siempre verifica AMBAS soluciones en la ecuación original.
1

Ejemplo 1 — Ecuación Básica de Valor Absoluto

|x| = 5.5
Resuelve: |x| = 5.5. Interpreta la solución gráficamente.
Caso + (Positivo)
x = 5.5|5.5| = 5.5 ✓
Caso − (Negativo)
x = −5.5|−5.5| = 5.5 ✓
Soluciones:
x = 5.5
x = −5.5
📐 Interpretación: Los dos puntos distan 5.5 unidades del cero: −5.5 y 5.5.
2

Ejemplo 2 — Valor Absoluto con Expresión

|2x − 7| = 5
Resuelve: |2x − 7| = 5
Caso + (Expresión = 5)
2x − 7 = 5Caso positivo
2x = 12+7 a ambos lados
x = 6÷2
Caso − (Expresión = −5)
2x − 7 = −5Caso negativo
2x = 2+7 a ambos lados
x = 1÷2
Soluciones:
x = 6
x = 1
✓ x=6: |2(6)−7|=|5|=5 ✓  |  x=1: |2(1)−7|=|−5|=5 ✓
3

Ejemplo 3 — Despejar Primero el Valor Absoluto

3|x + 4| − 2 = 13
Resuelve: 3|x + 4| − 2 = 13
3|x + 4| − 2 = 13Ecuación original
3|x + 4| = 15+2 a ambos lados
|x + 4| = 5÷3 ambos lados — ahora aplica los dos casos
Caso +
x + 4 = 5
x = 1−4
Caso −
x + 4 = −5
x = −9−4
x = 1
x = −9
✓ x=1: 3|5|−2=15−2=13 ✓  |  x=−9: 3|−5|−2=15−2=13 ✓
🔍

Monitoring Progress 1 — Resolver |expresión| = c

Parte 2 — Sin Solución / Solución Especial
|expresión| = c donde c < 0 → Sin Solución
El valor absoluto NUNCA puede ser negativo. Si c es negativo, la ecuación no tiene solución.
4

Ejemplo 4 — Sin Solución

|5x − 1| = −3
Resuelve: |5x − 1| = −3
❌ El valor absoluto siempre es ≥ 0. No puede ser igual a −3 (negativo).
Sin solución (no solution). No existe ningún valor de x que satisfaga la ecuación.
5

Ejemplo 5 — Despejar para revelar Sin Solución

2|x + 3| + 5 = 1
Resuelve: 2|x + 3| + 5 = 1
2|x + 3| + 5 = 1Ecuación original
2|x + 3| = −4−5 a ambos lados
|x + 3| = −2÷2 — ¡ojo! el resultado es negativo
❌ |x + 3| = −2 no tiene solución porque el valor absoluto no puede ser negativo.
🔍

Monitoring Progress 2 — Casos Especiales y Complejos

Parte 3 — Aplicaciones: Tolerancias y Errores
6

Ejemplo 6 — Temperatura de Horno / Tolerancia

Un horno debe estar a 350°F con tolerancia de ±15°F. ¿Cuáles son las temperaturas extremas aceptables?
|t − 350| = 15|temperatura − objetivo| = tolerancia
Caso +
t − 350 = 15
t = 365°F+350
Caso −
t − 350 = −15
t = 335°F+350
✓ La temperatura debe estar entre 335°F y 365°F para ser aceptable.
🔍

⚠️ Error Analysis
✗ Encuentra y corrige el error
|x − 4| = 7 x − 4 = 7 x = 11 ← Solo escribió un caso
Error: Solo consideró el Caso Positivo. Las ecuaciones de valor absoluto tienen DOS casos.
Caso +: x − 4 = 7 → x = 11
Caso −: x − 4 = −7 → x = −3
✓ Verifica: |11−4|=7 ✓ | |−3−4|=|−7|=7 ✓
