Parte 1 — Resolver con Suma y Resta
⭐ Core Concept — Propiedades de Igualdad (Suma y Resta)
Addition Property of Equality
Sumar el mismo número a ambos lados produce una ecuación equivalente.
Si a = b, entonces a + c = b + c
Subtraction Property of Equality
Restar el mismo número de ambos lados produce una ecuación equivalente.
Si a = b, entonces a − c = b − c
Recuerda: Para despejar x, aplica la operación inversa. Si hay +b, resta b de ambos lados. Si hay −b, suma b a ambos lados.
1

Parte 2 — Resolver con Multiplicación y División
⭐ Core Concept — Propiedades de Igualdad (Multiplicación y División)
Multiplication Property of Equality
Multiplicar ambos lados por el mismo número ≠ 0 produce ecuación equivalente.
Si a = b, entonces a · c = b · c  (c ≠ 0)
Division Property of Equality
Dividir ambos lados entre el mismo número ≠ 0 produce ecuación equivalente.
Si a = b, entonces a ÷ c = b ÷ c  (c ≠ 0)
Recuerda: Si x está multiplicado por 5, divide entre 5. Si x está dividido entre 3, multiplica por 3.
2

Ejemplo 3 — Velocidad promedio (Usain Bolt)

d = r · t
En los Juegos Olímpicos de 2012, Usain Bolt ganó los 200 metros en 19.32 segundos. ¿Cuál fue su velocidad promedio?
1. ComprendeDistancia = 200 m, tiempo = 19.32 s. Se busca: velocidad r.
2. PlanFórmula: d = r · t  →  200 = r · 19.32
3. Resuelve
200 = r · 19.32Ecuación
200/19.32 = rDivide entre 19.32
r ≈ 10.35 m/sSimplifica
4. RevisaA ~10 m/s × 20 s ≈ 200 m ✓ Razonable.

Ejemplo 4 — Cambio de temperatura (Spearfish, SD)

Modelado verbal
El 22 de enero de 1943, la temperatura en Spearfish bajó de 54°F a las 9:00 a.m. a −4°F a las 9:27 a.m. ¿Cuántos grados bajó?
Ecuación−4 = 54 − T  (T = grados de caída)
−4 − 54 = −TResta 54
−58 = −TSimplifica
T = 58°FMultiplica por −1
✓ Desde 54° a 0° son 54°, más 4° hasta −4°: 54 + 4 = 58 ✓
⚠️ Error Analysis
✗ Encuentra y corrige el error

Un estudiante resolvió −0.8 + r = 12.6 así:

−0.8 + r = 12.6
r = 12.6 + (−0.8) = 11.8 ← INCORRECTO
El error: El estudiante sumó −0.8 en lugar de sumarlo al otro lado.
Correcto: Sumar 0.8 a ambos lados → r = 12.6 + 0.8 = 13.4
✓ Verifica: −0.8 + 13.4 = 12.6 ✓
